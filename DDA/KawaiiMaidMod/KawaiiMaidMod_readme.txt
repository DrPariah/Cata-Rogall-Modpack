KawaiiMaidMod  230302版
　このMODは、KawaiiMaidMod v2.7.4を元に作成された「KawaiiMaidModの再配布バージョン（Ezsk氏による#10746確認版）」に、さらに不具合の修正・バランス調整をしたものです。
　安定版0.Gで動作確認済みです。
　このMOD単体で動作します。
　なお、オリジナルファイル(KawaiiMaidMod v2.7.4)に付属していたサンプル画像は、容量削減のため削除されています。

不具合の修正
・jsonエラーが出ないように修正。
・同じアイテムを複数スタックした場合に、アイテム名の後ろにsが付いてしまう不具合を修正。
・髪色、目の色、肌色がステータス画面の特質に表示されてしまう不具合（？）を修正。
・メイド服LSS-PRO と 温熱式ホワイトブリム と ホワイトブリムLSS-PRO をUPS対応にできない不具合を修正。
・弓のフラグに"WATER_FRIENDLY"と"WATERPROOF_GUN"と"NEVER_JAMS"を追加
・弓の"ammo_effects"に"NEVER_MISFIRES"を追加。
・弓に"pocket_data"を追加。
・弓のフラグから"STR_DRAW"を削除。
　　現在使用されていないもののようです。
・弓が ボウスリンガー に入らない不具合を修正。
・弓のMOD取り付け位置を修正。
・アイテムの修理難度が（類似のアイテムと比較して不自然に）高いものがある不具合を修正。
・アーク溶接機で修復可能な素材を追加するMODと競合する問題を修正。
　　A&M素材のアイテムは修復できなくなりましたが、その代わり近接攻撃で耐久が下がらなくなりました。
・copy-from の仕様変更（？）のため不要な部分を削除。
・一部の装備で動作制限が異常に高く表示される不具合を修正。
・SHELIAの格納時の不具合を修正。
・髪型が二重に定義されていた不具合を修正。
・腰積層廃止のため、多機能ベスト改の積層を上着に、シークレットポーチ(左)[kawaii]の積層を帯に変更。
・"hair1blondjp"の表示優先度が正しく指定されていなかった不具合を修正。

バランス調整
・クラフトに必要なターン数をかなり増加。
・メイド服をロングスカートにするレシピの必要なターン数を増加。
・ビキニパンツ(黒)[kawaii] と ビキニトップ(黒)[kawaii] をお洒落に。
・メイド服[kawaii] の性能を メイド服 と同じに。
・メイド服LSS-PRO の動作制限を 軽量型サバイバースーツ と同じに。
・メイド服Ex と メイド服LSS-PRO のポケットを 軽量型サバイバースーツ と同じに。
・厚手タイツ の被覆率を100、暖かさを20に。
・ハードケースシリーズ を色々調整。
　　ドットに合わせ、かなり長いものも入るように設定。
・シークレットポーチ のポケットの長さを設定。
・ハードケースを除く布製で material_thickness が1のアイテムの material_thickness を削除。（material_thicknessが0.1の扱いになるようです）
・布製で material_thickness が2のアイテムの material_thickness を0.2に変更。（ただし バラクラバ改[kawaii] は0.5）
・レシピに運動レベルを設定。
・いくつかの武器にカテゴリーを設定。これにより対応する武術が使えるように。
・被覆部位細分を設定。
　　これに伴いガーターベルト[kawaii]の被覆部位は脚ではなく胴体下部になりました。
・溶接に必要な遮光レベルが1になったため、ハイテク眼鏡の遮光レベルを1に。
・切断レベルの変更のため、レシピおよびアイテムの切断レベルを調整。
・SHELIA(格納)の分類と近接攻撃力を調整。
・ツインステーキナイフの重量と体積をステーキナイフのちょうど２倍に。
・ホワイトブリムLSS-PROからブロック性能を削除。
・靴下類の動作制限を0に。
・眼鏡類の積層を通常から肌着に。
・眼鏡類とハードケースが「着心地が悪い」にならないように。
・ホワイトブリムの積層を肌着から通常に。
・ガラスの弓へのMOD取り付けを可能に。
・アイテムに長さを設定。
・弓の必要筋力を増加。
・クリスタルツインダガーの攻撃力と体積を調整し、瞬撃を削除。
・武器を変形させるときは装備していなければならないように。
・いくつかの武器を刺突武器に。
・いくつかの武器を必ず両手で持つように。

既知の不具合
・NPCに髪色、目の色、肌色の特質が複数付いてしまうことがある。
・メイド服LSS-PROの電池関係の動作が怪しい。電池を外せないなど。
・靴を複数装備できる。

・必要技術が設定されていないレシピがある。
・メイド服をロングスカートにするレシピでスキル上げができる。
　　でも失敗しても知りませんよ？
